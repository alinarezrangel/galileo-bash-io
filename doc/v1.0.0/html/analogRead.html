analogRead

Uso

analogRead.sh pin

Descripción

Lee un valor analógico.

Argumentos

pin
Pin a leer. Debe haber sido declarado como entrada analógica.

Devuelve

Un valor de 12-bits que representa el valor del pin.

Ejemplos

Leer el valor de un pin

pinMode.sh A0 analog
pinMode.sh A1 analog
pinMode.sh A2 analog
echo "El pin esta en " $(analogRead.sh A0)
echo "El pin esta en " $(analogRead.sh A1)
echo "El pin esta en " $(analogRead.sh A2)

Errores

Si el pin no ha sido declarado como entrada analógica, su comportamiento esta indefinido.
