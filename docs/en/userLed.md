# `userLed.sh` - Controls the user's LED #

The command `userLed.sh` can enable, disable and write in the user's LED
(in Arduino, *user led*).

## Uso ##

```bash
userLed.sh enable # [1]
userLed.sh write $value # [2]
userLed.sh disable # [3]
```
